# ImageSelector
A graphical selection tool simulating Photoshop’s lasso selection and magic wand, implementing Dijkstra’s algorithm and a priority queue, achieving 98% precision verified by a self-developed test plan.

Contributors: Jin Chen, Ivan Dong

Programming Language: Java
